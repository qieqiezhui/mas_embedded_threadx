# BMI088 6 轴 IMU 模块文档

## 架构设计

### 数据流

```
┌──────────────┐     ┌──────────────┐
│  SPI2 (H7)   │     │  SPI2 (H7)   │
│  SPI1 (F4)   │     │  SPI1 (F4)   │
│  CS: PC3     │     │  CS: PC0     │
│  (Gyro)      │     │  (Accel)     │
└──────┬───────┘     └──────┬───────┘
       │ SPI Blocking       │ SPI Blocking
       ▼                    ▼
  _bmi088_read_reg()    _bmi088_read_reg()
  (dummy=1)             (dummy=2)
       │                     │
       ▼                     ▼
  ┌────────────────────────────────────────┐
  │           bmi088_device (静态全局)     │
  │                                        │
  │  acc[3]   (m/s²)    ← 6G 量程          │
  │  gyro[3]  (dps)     ← 2000dps 量程     │
  │  temperature (℃)                       │
  │  BMI088_Cali_Offset (标定数据)         │
  └────────────────────────────────────────┘
       ▲
       │ 直接读取 dev->acc / dev->gyro
       │
  ┌────┴──────────────────────────┐
  │         上层应用              │
  │  Module_BMI088_get_accel()    │
  │  Module_BMI088_get_gyro()     │
  │  Module_BMI088_get_temp()     │
  │  Module_BMI088_get_device()   │
  └───────────────────────────────┘
```

### `Bmi088_device_t` 设备结构

| 字段 | 类型 | 说明 |
|------|------|------|
| `initialized` | `uint8_t` | 初始化完成标志 |
| `gyro_device` | `SPI_Device *` | 陀螺仪 SPI 设备 |
| `acc_device` | `SPI_Device *` | 加速度计 SPI 设备 |
| `bmi088_pwm` | `PWM_Device *` | 温度控制 PWM |
| `pid_temp` | `PIDInstance` | 温度 PID 控制器 |
| `acc[3]` | `float` | 加速度 X/Y/Z (m/s²) |
| `gyro[3]` | `float` | 角速度 X/Y/Z (dps) |
| `temperature` | `float` | 芯片温度 (℃) |
| `BMI088_Cali_Offset` | `BMI088_Cali_Offset_t` | 标定偏移数据 |
| `BMI088_ERORR_CODE` | `BMI088_ERORR_CODE_e` | 错误码 |

### `BMI088_Cali_Offset_t` 标定数据

| 字段 | 类型 | 说明 |
|------|------|------|
| `AccelScale` | `float` | 加速度计缩放因子 (9.81 / gNorm) |
| `GyroOffset[3]` | `float` | 陀螺仪零偏 (dps) |
| `gNorm` | `float` | 标定测得的重力模长 |
| `TempWhenCali` | `float` | 标定时的芯片温度 (℃) |
| `Calibrated` | `uint8_t` | 标定完成标志 |

### 传感器配置

| 参数 | Accel | Gyro |
|------|-------|------|
| 量程 | ±6G | ±2000 dps |
| ODR | 800 Hz | 1000 Hz |
| 滤波器带宽 | Normal (OSR4) | 116 Hz |
| 灵敏度 | `BMI088_ACCEL_6G_SEN` (≈0.00179 m/s²/LSB) | `BMI088_GYRO_2000_SEN` (≈0.00107 dps/LSB) |
| 中断 | INT1 输出 DRDY | INT3 输出 DRDY |
| WHOAMI | 0x1E | 0x0F |

### 初始化流程

```
Module_BMI088_init()
  │
  ├── BSP_SPI_Device_Init() ×2   (Gyro CS + Accel CS)
  ├── BSP_PWM_Device_Init()       (温度控制 PWM)
  │
  ├── bmi088_acc_init()
  │     ├── SPI 模式切换 (I2C→SPI, fake read)
  │     ├── 软复位 + 延时 300ms
  │     ├── WHOAMI 校验 (0x1E)
  │     └── 写 6 个寄存器 → 回读校验
  │
  ├── bmi088_gyro_init()
  │     ├── 软复位 + 延时 300ms
  │     ├── WHOAMI 校验 (0x0F), 失败 → BMI088_NO_SENSOR
  │     └── 写 6 个寄存器 → 回读校验
  │
  ├── PIDInit()                   (温度 PID: Kp=20, MaxOut=1000)
  ├── BSP_PWM_Start()             (若 BMI088_TEMP_ENABLE=1)
  │
  └── 加载 Flash 标定数据
        ├── 魔数 0xAA 有效 → memcpy
        └── 无效 → 执行标定
```

### 标定流程

```
Module_BMI088_calibrate_bmi088_offset()
  │
  ├── LED 黄灯
  │
  └── do-while (直到零偏 + gNorm 满足阈值)
        │
        ├── 采样 6000 组 (acc + gyro)
        │     ├── 数据校验 (剔除 NaN/Inf/异常范围)
        │     ├── 计算 gNorm = sqrt(ax²+ay²+az²)
        │     ├── 累加求和 + 更新 min/max
        │     ├── 波动检测 → 设备移动 → 重置采样
        │     └── 超时保护 (≈7s) → 使用默认值
        │
        ├── 计算平均值: gNorm, GyroOffset[3]
        ├── 记录标定温度
        ├── 检查: |gNorm-9.81| ≤ 0.5 且 |offset| ≤ 0.01
        │
        └── 未通过 → 重新采样
  │
  ├── AccelScale = 9.81 / gNorm
  ├── 写入 Flash (魔数 0xAA)
  ├── 写入 dev->BMI088_Cali_Offset
  └── LED 绿灯
```

### 标定稳定性阈值

| 参数 | 值 | 说明 |
|------|-----|------|
| `CALI_GNORM_DIFF_MAX` | 0.5 | gNorm 最大波动 |
| `CALI_GNORM_TOLERANCE` | 0.5 | gNorm 与 9.81 允许偏差 |
| `CALI_GYRO_DIFF_MAX` | 0.15 dps | 陀螺仪最大波动 |
| `CALI_GYRO_OFFSET_MAX` | 0.01 dps | 零偏合格上限 |
| `CALI_TARGET_SAMPLES` | 6000 | 目标采样数 |

## 注意事项

1. **标定时设备必须水平且静止** — 标定期间若检测到 gNorm 波动 >0.5 或陀螺仪波动 >0.15 dps，采样自动重置。LED 黄灯表示标定进行中。

2. **首次上电自动标定** — Flash 中无有效标定数据时，`Module_BMI088_init()` 会阻塞执行标定（约 6 秒），确保设备静止。

3. **数据单位** — `acc` 输出 m/s²（已应用量程转换和标定缩放），`gyro` 输出 dps（已应用量程转换和零偏修正）。
